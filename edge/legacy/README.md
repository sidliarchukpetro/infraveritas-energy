# Legacy edge mock

Не використовує: ATECC608B, GPS NEO-6M, DC-side PZEM, tamper switches.

**Не для нової розробки.** HAL з MockEdgeDevice і RaspberryPiEdgeDevice пишемо у `edge/hal/` згідно з Етапом 4a плану v1.4.
